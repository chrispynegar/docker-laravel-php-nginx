# Laravel PHP and Nginx Image

A base image running PHP and Nginx built for running a Laravel application.

```sh
$ docker pull chrispynegar/laravel-php-nginx
```

### Available Aliases

| Alias | Command |
| ----- | ------- |
| ```art``` | ```php artisan``` |
| ```phpunit``` | ```php ./vendor/bin/phpunit``` |
| ```pu``` | ```php ./vendor/bin/phpunit``` |
